# Information Interaction Structures (IIS)

This repository contains a reproducible implementation of the Information Interaction Structures (IIS) framework for identifying higher-order dependence patterns using empirical copulas and Möbius decomposition.

## Repository Structure

benchmark_datasets/
    Input CSV datasets

src/
    Core analysis modules

results/
    Output tables and summaries

IIS_pipeline.py
    Main pipeline

requirements.txt
    Python dependencies
